## Overview
MongoDB is a NoSQL database that stores data as flexible JSON-like documents instead of rows and tables like traditional SQL databases.

Built for modern applications that need to handle large amounts of unstructured or frequently changing data.

## Prerequisites
- Node.js installed
- MongoDB Atlas account

## Setup Steps

### Step 1: Install MongoDB
```bash
sudo apt install mongodb
```

## Common Issues
| Problem  |    Fix   |

| Auth failed   | Check username/password in connection string   |
